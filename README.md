# R6-Stats-Discord-Bot
Discord.py bot to track rainbow six siege stats.

**To use you must create a config.json file in the same directory as the R6StatsBot.py file. In the json file you must include the following:**

```
{
    "token": "your token",
    "prefix": "prefix of your choice"
}
```

**PLEASE NOTE YOU MUST CHANGE THE TOKEN AND PREFIX FIELDS TO YOUR OWN VALUES**
**YOU CAN OBTAIN A TOKEN HERE: https://discord.com/developers/applications**
